## Setup

Install the dependencies:

```bash
yarn install
```

## Development

Start the development server on `http://localhost:3000`

```bash
yarn run dev
```

### Linting

```bash
yarn run lintfix
```

## Production

Vercel deploys the current latest tag automatically. Should another tag or a revert be needed, contact @aaronsteed
